## Judul Project
Lifepedia: Platform Berbagi Tutorial dan Lifehack Berbasis Web

## Deskripsi Singkat Project
Lifepedia merupakan aplikasi web berbasis komunitas yang memungkinkan pengguna untuk membagikan tutorial, tips, dan lifehack dalam berbagai kategori. Pengguna dapat membuat, membaca, mengedit, dan menghapus tutorial yang telah diposting, serta melihat riwayat tutorial yang telah dibagikan berdasarkan akun pengguna.

## Penjelasan Singkat Database
Database Lifepedia dirancang untuk mengelola data pengguna, kategori tutorial, tutorial yang diposting, serta bookmark tutorial. Setiap pengguna dapat membuat lebih dari satu tutorial dan menyimpan tutorial yang dianggap bermanfaat melalui fitur bookmark. Setiap tutorial dikelompokkan ke dalam satu kategori tertentu, sehingga relasi antar tabel dapat mendukung pengelolaan akun pengguna, kategori tutorial, publikasi tutorial, dan penyimpanan tutorial oleh pengguna.
