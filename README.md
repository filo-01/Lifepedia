# Deskripsi Singkat Project
Lifepedia merupakan aplikasi web berbasis komunitas yang memungkinkan pengguna untuk membagikan tutorial, tips, dan lifehack dalam berbagai kategori. Aplikasi dibangun menggunakan Node.js, Express.js, Express Handlebars, dan SQLite. Pengguna dapat melakukan registrasi, login, membuat tutorial, melihat tutorial, mengedit, menghapus, memberikan komentar, menyimpan tutorial ke bookmark, serta mengelola profil pengguna.

# Cara Menjalankan Aplikasi
1. Clone repository GitHub.
2. Install dependency.

```bash
npm install
```

3. Inisialisasi database.

```bash
npm run db:init
```

4. Isi database dengan data awal (Seeder).

```bash
npm run db:seed
```

5. Jalankan aplikasi.

```bash
npm run dev
```


6. Buka browser.

Local:

```
http://localhost:3000
```

Deployment:

```
https://lifepedia-0bvmhpz1.b4a.run
```
